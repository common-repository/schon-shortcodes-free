Provide many shortcodes for the free version of the theme Schon.

== Description ==

This plugin enhance the features of the Free WooCommerce theme Schon. It provides many shortcodes:

* Socials icons list
* Menu for logged/not-logged users
* Products search form
* Info box
* Custom button
* Custom title

== Installation ==
1. Install and active the plugin
2. That's all, you can already use the shortcodes

You can more specific details on them in the theme documentation
